快读，快写

2020-02-27

快读为什么叫快读？
因为它快呀。（废话
它为什么快呢？
因为它在读入的时候没有缓冲（网上这么说的
下面直接给出它的代码。

#include <bits/stdc++.h>
using namespace std;
typedef (long long) type;
inline type read(type &n)
{
    static int f;
    static char c;
    n = 0;
    f = 1;
    c = getchar();
    while (c < '0' || c > '9')
    {
        if (c == '-') f = -1;
        c = getchar();
    }
    while(c >= '0' && c <= '9')
    {
        n = (n << 3) + (n << 1) + (c ^ '0');
        c = getchar();
    }
    n *= f;
}
void write(type n)
{
    if (n / 10 == 0) putchar((n % 10) | '0');
    else
    {
        write(n / 10);
        putchar((n % 10) | '0');
    }
}
type n;
int main()
{
    while (true)
    {
        read(n);
        if (n < 0)  //为了减少判断次数，在输出的时候进行判断
        {
            putchar('-');
            write(-n);
        }
        else
            write(n);
        putchar('\n');
    }
}

typedef后面的long long可以换成int。
在上面的代码中我用了位运算，下面我们来讲一下为什么可以使用为运算.
