集合的含义与表示

定义

  • 元素：研究对象称为元素
  • 集合：一些元素的总体（简称 集）

含义

集合的表示通常使用大写的拉丁字母A，B，C，···
如果a是A集合的元素，就说a属于A，记作a ⁣∈ ⁣Aa\!\in\!Aa∈A
如果a不是A的集和元素，就说a不属于A，记作a ⁣∉ ⁣Aa\!\notin\!Aa∈/​A

数学中的常用集合

  • N：非负整数集（或自然数集）
  • N+或N∗N_+ 或 N^*N+​或N∗：正整数集
  • Z：整数集
  • Q：有理数集
  • R：实数集

描述

列举法

例：方程(x−1)(x+2)=0(x-1)(x+2)=0(x−1)(x+2)=0的所有实数根可以表示为{1,−2}\{1, -2\}{1,−2}
像这样把集合元素一一列举出来，并用花括号“{ }\{\ \}{ }”括起来表示集合的叫 列举法

描述法

例1：表示x−7<3x-7<3x−7<3的解集
D={x ⁣∈ ⁣R∣x<10}D=\{x\!\in\!R|x<10\}D={x∈R∣x<10}
例2：表示任何所有奇数的集合
E={x ⁣∈ ⁣Z∣x=2k+1,k ⁣∈ ⁣Z}E=\{x\!\in\!Z|x=2k+1, k\!\in\!Z\}E={x∈Z∣x=2k+1,k∈Z}
像这样用元素的共同特征表示的方法叫 描述法
